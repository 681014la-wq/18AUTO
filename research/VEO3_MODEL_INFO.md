# Veo 3 — Google DeepMind 영상 생성 모델 정보

> 원본: https://github.com/veo-3/veo-3
> 공식: https://deepmind.google/models/veo/
> 프롬프트 & 데모: https://aiprompts.video/veo3

---

## 1. 개요

Veo 3는 Google DeepMind의 최신 AI 영상 생성 모델.
Google I/O 2025에서 발표.
텍스트 또는 이미지 프롬프트로 고품질 1080p 영상 생성.

---

## 2. 핵심 기능

| 기능 | 설명 |
|------|------|
| Multimodal Input | 텍스트 또는 이미지 프롬프트로 영상 생성 |
| High-Quality Output | 1분 이상, 1080p 해상도 |
| Audio Integration | 대화, 효과음, 배경음 등 네이티브 오디오 |
| Advanced Realism | 실세계 물리 시뮬레이션, 정확한 립싱크 |
| Narrative Understanding | 복잡한 프롬프트로 일관된 스토리라인 |

---

- 미국 내 Gemini 앱 (AI Ultra 플랜 $249.99/월)
